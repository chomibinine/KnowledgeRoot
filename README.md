# 🌱 Knowledge Root (지식뿌리)
암기하지 마세요, 뿌리부터 이해하세요. 
데이터 기반 통합 지식 포털 'Knowledge Root' 입니다.

## 🚀 Services
* **VocaRoot:** 접두사, 어근, 접미사 분해 알고리즘 기반 영단어장
* **HanjaRoot:** 한국어문회 8급~1급 부수 및 어원 분석 한자 학습
* **IdiomRoot:** 수능 국어 필수 사자성어 및 고사 유래 분석

## 📁 Folder Structure
* `/` : 메인 랜딩 페이지
* `/lang/voca/` : 영단어 (VocaRoot)
* `/lang/hanja/` : 한자 (HanjaRoot)
* `/lang/idiom/` : 사자성어 (IdiomRoot)
